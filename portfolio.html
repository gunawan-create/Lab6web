Tentang Saya

Ali Gunawan

Saya adalah mahasiswa Universitas Pelita Bangsa yang memiliki minat besar dalam dunia teknologi, terutama di bidang pengembangan web dan desain antarmuka (UI/UX). Saya suka mempelajari framework modern seperti Bootstrap dan React untuk menciptakan tampilan yang interaktif dan responsif. Selain itu, saya juga tertarik dalam pengembangan sistem berbasis database serta logika pemrograman yang efisien.

Di luar kegiatan akademik, saya gemar mempelajari tren teknologi terbaru, mendesain layout website, serta mengasah keterampilan front-end agar mampu menciptakan pengalaman pengguna yang nyaman dan menarik.
